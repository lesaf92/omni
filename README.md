# OMNI: Omnidirectional Motion when Navigating Indoors 

Code for the OMNI robot

## Hardware
- Intel Realsense D435i depth camera
- Raspberry Pi 4B
- Arduino Mega 2560
- LSM9DS1 IMU
- 4x stepper motors
## Software
- Ubuntu 16.04
- ROS Kinetic
